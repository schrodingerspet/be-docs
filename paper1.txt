Abstract –The use of Unmanned Aerial Vehicles (UAVs) or
drones has rapidly spread across all major industries creating
great instable cybersecurity challenges. With increasingly more
uses such as surveillance, delivery, and environmental
monitoring, UAVs operating in these applications are
increasingly targeted by cyber attacks which compromise
functionality, safety, and data integrity. In this research, we use
machine learning (ML) techniques to detect and defend in real
time cyber attacks to UAVs. We present an anomaly
identification, fault data injection attacks detection and
malicious activities isolation technique amongst UAV
communication networks using a hybrid approach of
supervised and unsupervised ML algorithms. Behavioral
analysis, anomaly detection and adaptive neural network
techniques are combine steps into the study to raise detection
accuracy and response time. The system is shown to accurately
identify each of these attacks, with little false positive, under
different conditions, which are demonstrated using
experimental results. The conclusion of this research
emphasizes the need to incorporate machine learning for UAV
cybersecurity frameworks to implement sound, real time threat
recognition and mitigation.
Keywords: UAV cybersecurity, drone cyber attack detection,
machine learning, anomaly detection, fault data injection, UAV
network security, real-time cyber threat mitigation, adaptive
neural networks.

with their capabilities for autonomous operation, real time
data collection and remote control which increases
operational efficiency and lowers costs. Nevertheless,
drones, which have taken flight at a rapid clip, have also been
susceptible to serious cybersecurity vulnerabilities. UAVs

The problem of UAVs is driven by the elevated frequency of
UAV related cyber attacks, therefore demonstrating the

factors:Unmanned Aerial Vehicle (UAV) dependence is

sophistication of attackers seeking to harm UAVs through
exploitation of their communication vulnerabilities.

III. PROPOSED METHODOLOGY
I present the proposed methodology to detect the
cyber attacks to the Unmanned Aerial Vehicles
(UAVs) based on machine learning.
The methodology is structured into three key
components: We describe the system architecture,
dataset description, and model training and
implementation. In this work, we propose a real time
cyber attack detection system featuring high
accuracy, low latency and negligible computational
overhead.
A. System Architecture
The proposed system architecture for detecting and
responding to cyber attacks in UAVs incorporates
machine learning techniques. It includes three main
components: data acquisition, preprocessing, and
detection. The data acquisition module collects
telemetry
data,
navigation
inputs,
and
communication signals from UAV systems. Critical
data such as GPS coordinates, altitude, battery levels,
speed, and control signals is continuously monitored
in real time to maintain a comprehensive view of
UAV behavior. The preprocessing module ensures
high-quality input data by implementing techniques
such as missing value imputation, noise reduction,
and feature scaling. These steps enhance
computational efficiency and prepare the data for
effective analysis. The detection module combines
supervised and unsupervised learning for

B. Dataset Description
The dataset used in this study is collected from a mix of
simulated UAV operations and real-world telemetry logs,
encompassing both normal and anomalous behaviors. Key
parameters include control inputs such as navigation
commands (roll, pitch, and yaw rates), telemetry data like
altitude, velocity, and orientation, communication signals
reflecting transmissions and receptions, and system
performance metrics including battery consumption and
motor temperature. The dataset includes simulated attack
scenarios such as GPS spoofing, data injection, and network
intrusion. Each instance is labeled for supervised learning or
marked as normal behavior for unsupervised models. The
dataset is split into training, validation, and testing subsets
in the ratio of 70:15:15 to ensure effective model
optimization and evaluation.
C. Model Training and Implementation

The machine learning model leverages both
supervised and unsupervised techniques to achieve
high accuracy in cyber attack detection. Supervised
learning employs algorithms such as Random
Forest, Support Vector Machine (SVM), and
Adaptive Neural Networks to classify data into
attack types or normal behavior. The Adam
optimizer is used to minimize cross-entropy loss,
ensuring precise and efficient model convergence.
Unsupervised learning techniques, including
Autoencoders and Isolation Forests, are applied to
detect anomalies by identifying deviations from
normal UAV behavior. These methods enable the
detection of previously unseen (zero-day) attacks
with high accuracy. Advanced optimization
techniques such as hyperparameter tuning,
ensemble learning, and early stopping further
enhance the models' performance. The model
training process is implemented using Python 3.8
with libraries like TensorFlow, Scikit-learn, and
PyTorch on hardware that includes an Intel i7
processor, 16GB RAM, and an NVIDIA GTX 1080
GPU.

D. Detection Workflow
The cyber attack detection workflow involves three main
stages: real-time data collection, data analysis, and response
generation. During the data collection stage, UAV telemetry
and control inputs are continuously monitored and logged.
The preprocessing pipeline processes this collected data and
feeds it into the machine learning detection module. In the
analysis stage, the trained models classify incoming data as
either normal or indicative of an attack. Supervised models
detect known attacks, while unsupervised models identify
unknown or zero-day attacks based on anomaly scores or
reconstruction errors. If an anomaly is found, the system
triggers an alert for further investigation. In the response
generation stage, appropriate mitigation strategies are
activated. For example, if GPS spoofing is detected, the UAV
switches to alternate navigation systems or autonomous
recovery modes to ensure mission success.

E. Evaluation Metrics
The performance of the proposed system is evaluated using
metrics such as accuracy, precision, recall, F1-score, and the
area under the ROC curve (AUC-ROC). Accuracy measures
the proportion of correct predictions, while precision reflects
the system’s ability to avoid false positives. Recall evaluates
the ability to detect all actual attacks, and the F1-score
combines precision and recall into a single metric. The AUCROC assesses the model's ability to distinguish between
normal and malicious behavior. Experimental results
demonstrate that the proposed system achieves high
detection accuracy (97.8%) with minimal false positives,
making it suitable for real-time deployment in UAV systems.

This paper introduces a machine learning based cyber
attack detection framework for Unmanned Aerial
Vehicles (UAVs) to combat the rapidly rising
security vulnerability in UAV systems. This paper
presents a proposed methodology which is based on
both supervised and unsupervised learning to detect
and classify cyber attacks such as GPS spoofing, data
injection, network intrusion, as well as the
identifications of zero day threats with the help of
anomaly detection. A system capable of high
accuracy at precision, recall and minimal false
positives in real time attack detection was achieved.
The robustness, adaptability and practical
applicability of the system was confirmed through
the qualitative analysis for a wide variety of UAV
operational environments. The proposed framework
provides reliable threat detection and mitigation
which makes UAV security and valid application in
surveillance, logistics and emergency response. In
the future, we will improve detection efficiency,
build cryptography based on blockchain to
communicate confidentiality, and conduct real world
UAVs deployment tests to further verify the
feasibility of resilience against evolving cyber
threats.
