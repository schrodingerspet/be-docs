# Data Fusion & Feature Selection (SHAP)

Understanding why network packets aren't enough, and how SHAP proves physical telemetry catches stealthy attacks.

<Callout type="insight">
**What is a Shapley Value?** It is a game-theoretic approach to explain exactly which data points convinced the AI to flag an attack. In the Hassler et al. study, SHAP values guarantee explainability of the Black-box machine learning models.
</Callout>

### SHAP Analysis Highlights

The Hassler et al. study utilizes game-theoretic SHAP values to highlight the top 20 features for their SVM and FNN models. Notice how physical features like `flight_time` and `battery` rank alongside cyber features like `wlan.duration`.

<Accordion title="Why Physical Features Matter">
When a sophisticated attacker spoofs network traffic, the cyber features might look entirely legitimate to a firewall. However, if a drone is receiving "fly forward" commands but its physical `pitch` and `mp_distance_z` show it falling out of the sky, the anomaly is immediately obvious. SHAP analysis proved that the AI heavily weighted these physical mismatches.
</Accordion>

> "To the best of our knowledge, this is the first open source dataset generated by executing cyber attacks against a UAV and capturing both cyber and physical features simultaneously." — *Hassler et al.*
